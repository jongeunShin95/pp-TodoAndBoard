### 프로젝트 구조
```sh
├─src
│  ├─components/ # 화면 컴포넌트 모아둠
│  ├─containers/ # 컴포넌트를 뿌려주는 컨테이너 모아둠
│  ├─modules/ # 리듀서, 액션 함수 등
│  └─App.tsx
```
